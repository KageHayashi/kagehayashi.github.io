APT Backdoor

SSH Backdoor

Add key into ~/.ssh folder

  1. ssh-keygen
  2. write id_rsa.pub into ~/.ssh/authorized_keys
  3. set the right permission, 700 for ~/.ssh, 600 for authorized_keys, and 600 for id_rsa
